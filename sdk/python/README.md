# ang-sdk

Python client SDK generated by ANG.

## Install

```bash
python -m pip install -e .
```

## Quick Start

```python
from ang_sdk import AngClient

client = AngClient(base_url="http://localhost:8080", token="<jwt>")
# result = client.some_method(...)
client.close()
```
